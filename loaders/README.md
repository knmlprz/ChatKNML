# Loaders Module

This module contains data loaders for the ChatKNML chatbot.

## Installation

To install the loaders module, run the following command:

```bash
poetry install
```

## Testing

To run tests, run the following command:

```bash
poetry run pytest
```

## Formatting and linting

Formatting:

```bash
poetry run black .
```

Linting

```bash
poetry run flake8
```
